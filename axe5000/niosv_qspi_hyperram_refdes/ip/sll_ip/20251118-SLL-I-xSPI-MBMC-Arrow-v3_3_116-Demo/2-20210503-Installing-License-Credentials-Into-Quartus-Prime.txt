   1)  Floating Node License Credential 

       Please install (copy and paste) the entire License Credential with Embedded License Key 
       into Quartus Prime, then make neccasry adjustments to the License Key for your FlexLM server configuration.  
  
        Learn more about setting up a floating license in:
           Section 4.4 Configuring the License Manager Server
           https://www.intel.com/content/dam/www/programmable/us/en/pdfs/literature/manual/quartus_install.pdf

         You can also find a video on debugging your installation of a License Manager Server setup here:
           https://www.youtube.com/watch?v=llLaWatUbe8

    2)  Fixed Node License Credential 

        Please install (copy and paste) this entire License Credential with Embedded Fixed Node License Key into 
        the License .dat file for Quartus Prime on the computer compiling this design
		
	      For more details, read: 
            2-20210505-SynapticLabs-MBMC-Installation-FixedNode-StepByStepGuide.pdf
